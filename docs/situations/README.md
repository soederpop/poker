# Situation Documents

Save reusable study spots here.

Recommended rules:
- one spot per file
- use lowercase kebab-case filenames
- keep hero-first ordering for `stacks` and `positions`
- use semicolons or newlines in `actionHistory`
- prefer including both a frontmatter `title` and a matching markdown H1

Minimum schema:
- `title`
- `stage`
- `heroCards`
- `potSize`
- `toCall`

Common optional fields:
- `board`
- `stacks`
- `positions`
- `actionHistory`
- `villain`
- `stakes`

Template:
- copy `docs/situations/_template.md`

Examples in this folder:
- `btn-vs-bb-single-raised-pot.md`
- `turned-flush-draw.md`
