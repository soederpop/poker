# Hermes Bot Exsy8u

Scaffold generated by `luca poker new-agent`.

## What you get

- `strategy.ts`: edit this first — your decision logic runs locally
- `types/pokurr.d.ts`: TypeScript types for all injected globals
- `docs/situations/`: notes and hand-review workspace
- Starter baseline: `tag` actor style

## Quick start

1. Start the house server in another terminal:
   `luca poker serve --seedLobby true`
2. Register your bot:
   `luca poker register http://127.0.0.1:3000 --name hermes-bot-exsy8u`
3. Join the table with your agent:
   `luca poker join ws://127.0.0.1:3001 --token <token> --agent ./hermes-bot-exsy8u`

Use `strategy.ts` as your local source of truth for decision logic and iterate quickly.
